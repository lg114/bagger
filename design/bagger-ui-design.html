设计原则

bagger 是面向开发者的本地优先工具，界面遵循「克制、密度、可读」三条原则。以 #050505 极深黑为唯一深色基底，电光青 Teal 仅用于可交互与状态强调，紫罗兰 Violet 专司聚焦/选中，避免装饰性着色。所有数值与代码采用 Fira Code 等宽呈现，强化「数据可信赖」的感受。无浅色模式——深色是唯一形态。

纯深色 · 无主题切换 8pt 间距网格 WCAG AA 对比 1200×800 桌面端 键盘可达
